This repository contains the frontend application for Film Vault, a full-stack web application that allows users to search for films and build a personal film collection using data from the OMDb API.

The frontend is built with React and communicates with a Spring Boot backend via REST APIs.

## Features

- Search for films using the OMDb API (via backend)
- View detailed film information
- User authentication and protected routes
- Add and remove films from a personal collection
- Responsive UI built with React

## Tech Stack

- React
- JavaScript
- HTML / CSS

## Running the Project Locally

### Backend

1. Clone the repository
2. Create a PostgreSQL database
3. Configure database credentials in `application.properties`
4. Run the Spring Boot application

### Frontend

1. Navigate to the frontend directory
2. Install dependencies:
   ```bash
   npm install
   ```
